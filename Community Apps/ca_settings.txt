Hide Deprecated Applications=隐藏已弃用的应用
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=此设置将在可用应用列表中隐藏已弃用的应用程序, 但已弃用的应用仍将显示在已安装的应用和以前的应用列表中
Hide Incompatible Applications=隐藏不兼容的应用
This setting will hide applications that are listed as being incompatible with your version of Unraid=此设置将隐藏与您的 Unraid 版本不兼容的应用程序
Allow install of second instance=允许安装第二个应用
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=此选项会在卡片和弹出窗口上启用一个新按钮,该按钮允许您使用默认模板值重新安装已安装的应用程序。此选项有时可能会造成混淆，通常仅在您希望安装并同时运行两个相同的应用程序时才使用。.
Automatically open the sidebar=自动打开侧边栏
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=鼠标悬停在应用程序图标上时, 允许社区应用中心自动打开侧边栏以显示更多信息
Allow CA to check for updates to applications=允许社区应用中心检查应用程序的更新
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=此设置使社区应用中心能够检查应用程序的更新, 请注意：社区应用中心将始终检查自身的更新
Disable Mouse Wheel Page Changes=关闭鼠标滚轮翻页
This setting disables using left or right on a mouse wheel to change the page=此选项用于关闭鼠标左/右方向滚轮翻页
Mouse wheel threshold=鼠标滚轮阈值
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=当你将鼠标滚轮向左或向右移动时，浏览器会返回相应的变化值。请将这个变化值设置为阈值。需要注意的是，不同的浏览器和操作系统可能会返回不同的数值。
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=这是一个对所有设备都适用的全局设置。如果出现问题，请禁用鼠标滚轮翻页功能。
Allow CA to send any emergency notifications=允许社区应用中心发送紧急通知
This setting allows CA to periodically scan your system and alert you to the presence of=此设置允许社区应用中心定期扫描您的系统，并向您提醒系统中存在的问题
1 Known malware and security vulnerabilities within applications=1 应用程序中已知的恶意软件和安全漏洞
2 Critical security vulnerabilities within Unraid OS=2 Unraid OS 系统中的严重安全漏洞
3 Any installed applications which may seriously hinder the operation of your server=3 任何可能严重阻碍服务器运行的已安装的应用程序
Save CA debugging information=保存社区应用中心的调试信息
Only enable this when advised to from the author of Community Applications=仅当社区应用程序的开发者建议时才启用此功能
Enable developer mode=启用开发者模式
This unlocks some extra features within CA generally only used by developers Not recommended to enable=这会解锁社区应用中心中的一些额外功能(通常仅由开发人员使用, 不建议启用)
